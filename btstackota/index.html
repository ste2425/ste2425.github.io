BluePad32 Webapp Example

This app connects to an ESP32 running the BluePad32 ESP32 example code. Click the connect button to attempt to connect to the ESP. You can also connect a controller, the ESP will communicate with both at the same time.

Why not send r, g or b and see what happens to a connected DS4.

You can read values from and send values to the ESP. Data will be logged below.
